## Run Locally

Clone the project

```bash
  git clone https://github.com/adidongs1/slicing-with-react-js.git
```

Go to the project directory

```bash
  cd slicing-with-react-js
```

Start the server

```bash
  npm run dev
```

Get the local address link
